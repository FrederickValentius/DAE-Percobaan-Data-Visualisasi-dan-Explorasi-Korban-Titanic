# Titanic - Data Visualisation
Notebook ini berisi proses Exploratory Data Analysis (EDA) pada dataset Titanic untuk memahami faktor–faktor yang mempengaruhi keselamatan penumpang. Analisis dilakukan dengan memeriksa karakteristik data, distribusi variabel, dan hubungan antar fitur terhadap variabel Survived.

## Tujuan Analisis

memahami struktur data Titanic

mengidentifikasi variabel penting

melihat pola survival berdasarkan atribut penumpang

menjadi dasar untuk modeling machine learning

## Proses Analisis

Tahapan utama yang dilakukan:

1. Import dan Load Data

2. Pengecekan Missing Values

3. Descriptive Statistics

4. Explorasi tiap variabel seperti

  a. usia

  b. jenis kelamin

  c. kelas (Pclass)

  d. embarkation

5. Visualisasi distribusi data

6. Analisis hubungan variabel dengan Survival

## Temuan Utama

Beberapa pola penting dari hasil eksplorasi:

penumpang wanita lebih banyak selamat dibanding pria

kelas lebih tinggi (Pclass 1) memiliki tingkat survival tertinggi

anak–anak memiliki peluang selamat lebih tinggi daripada dewasa

jumlah penumpang laki–laki kelas 3 adalah yang paling banyak meninggal

## Kesimpulan

Karakteristik penumpang berpengaruh terhadap peluang keselamatan. Faktor paling menonjol adalah:

Gender

Pclass

Age
